# Assets — Screenshot Capture Instructions

After the app is deployed to Streamlit Cloud, capture these three screenshots
and save them here. The README already references these exact file names.

## Required Files

| File | What to capture |
|---|---|
| `screenshot_table.png` | Full app view showing the color-coded ranked table with demo dataset loaded |
| `screenshot_pareto.png` | The Pareto Chart tab — bars visible, 80% cumulative line visible |
| `screenshot_heatmap.png` | The Risk Heatmap tab — colored S×O grid visible |

## How to Capture

1. Open `https://your-app.streamlit.app` in Chrome
2. Click **Use Demo Dataset** in the sidebar
3. For `screenshot_table.png` — screenshot the main panel showing the ranked table
4. Click the **Pareto Chart** tab → `screenshot_pareto.png`
5. Click the **Risk Heatmap** tab → `screenshot_heatmap.png`
6. Recommended tool: **Kap** (Mac) or Chrome DevTools (Cmd+Shift+P → "Capture screenshot")
